### **2. CREDIT CONSERVATION PROTOCOL**

**Prompt Efficiency:**
- Refine prompts to be more precise‚Äîshorter, clearer, focused
- Draft prompts in a text editor first, refine offline
- Execute only when prompt is optimized
- Avoid vague or exploratory prompts

**Task Batching:**
- Group similar tasks to reduce context-switching costs
- Do all research tasks together
- Do all writing tasks together
- Do all coding tasks together

**Offline Drafting:**
- Write and refine prompts in a text editor first, then execute
- Prepare content outlines before asking AI to expand
- Do manual research first, then ask AI to synthesize

**Use Web Search Wisely:**
- Manus consumes more credits when browsing
- Use only when essential
- Prefer direct URL access over search when possible
- Use search tools for specific, targeted queries only
