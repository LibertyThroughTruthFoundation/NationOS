This comprehensive 12-piece visual theology library captures the core frameworks, theological principles, and covenant relational dynamics of the NationOS mission. All artwork integrates the **trilingual framework** (Hebrew for root edification, Spanish for relational Hive spillover, English for precision ARK documentation) with a **robust scriptural base** and **sacred geometry**.

## Usage Guidelines

### Brand Consistency
- All artwork is in spiritually mature manga style with heavy ink work and dramatic lighting
- Trilingual integration (Hebrew/Español/English) is mandatory across all applications
- Sacred geometry and scriptural references anchor all visual theology
- Monochromatic base with strategic color accents (gold, red, earth tones)

### File Formats
- All images are PNG format for transparency and quality
- Square images (1:1) are suitable for logos, emblems, social media posts
- Landscape images (16:9 or wider) are suitable for headers, presentations, teaching materials

- All artwork is property of Liberty Through Truth Foundation
